Optimization of Walking Gait on Quadruped Robots

The walking gait for simulated quadruped robots is solved by using a Genetic Algorithm as Optimizer. The whole process is simulated on pybullet's physics engine. The population has 8 robots, of 8 joints per robot. The genome for each robot defines the trajectory per each joint of a robot. The solution is to find / optimize the proper gait sequence in order to allow the robot to move forward ( +X ) by performing a walking pattern.
